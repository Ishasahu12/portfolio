# Isha Portfolio

A minimal, light-themed UX portfolio focused on real-world problem solving and human-centered design.

## Structure

```
├── index.html              # Home page
├── case-study-moneyship.html
├── case-study-creative.html
├── case-study-eco.html
├── css/
│   └── style.css          # Light theme, DM Sans, minimal shadows
├── js/
│   └── main.js            # Smooth scroll, animations
└── assets/
    └── texture.jpg         # Background texture
```

## Design Decisions

- **Theme**: Light (clean, approachable, human)
- **Font**: DM Sans (modern, readable, not generic)
- **Shadows**: Minimal (spacing creates structure)
- **Hero**: Subtle guitar (identity marker, not decoration)
- **Layout**: Hybrid — home page + separate case study pages
- **Positioning**: "I understand real users and design simple, usable products"

## Case Study Format

Each project follows: Problem → User Pain → Insight → Solution → Impact

## Deployment

Live: https://esha-portfolio-2026.pages.dev

Built with Cloudflare Pages.

## Customization

Edit `index.html` for:
- Project cards (in #work section)
- About content (in #about section)
- Contact links (in #contact section)

Resume button links to `resume.pdf` (add your file to root).
